# تحديثات سكيما المنتج - عرض الكمية مع الألوان والمقاسات

## 📋 التغييرات المنفذة

### 1. تعديل سكيما المنتج (schemaTypes/product.js)

#### التغييرات في حقل `inventory`:
- **تغيير من `colors` (array) إلى `color` (string)**: الآن يتم اختيار لون واحد فقط لكل متغير
- **تغيير من `sizes` (array) إلى `size` (string)**: الآن يتم اختيار مقاس واحد فقط لكل متغير
- **إضافة وصف لحقل `stock`**: "أدخل الكمية المتوفرة من هذا اللون والمقاس"
- **تحديث preview**: يعرض الآن اللون والمقاس بشكل فردي بدلاً من القوائم

**الفائدة**: 
- تبسيط إدخال المخزون
- كل توليفة (لون + مقاس) لها كمية محددة
- سهولة إدارة المخزون

### 2. تعديل مكون ProductDetails.js

#### أ. تصفية الألوان والمقاسات المتاحة فقط:
```javascript
// استخراج الألوان والمقاسات التي لها مخزون > 0
const colorsWithStock = inventory
    .filter(v => v.stock && v.stock > 0)
    .flatMap(v => v.color || []);

const sizesWithStock = inventory
    .filter(v => v.stock && v.stock > 0)
    .flatMap(v => v.size || []);
```

#### ب. عرض الكمية بجانب كل لون:
- حساب إجمالي الكمية لكل لون (عبر جميع المقاسات)
- عرض الرقم باللون الأخضر تحت اسم اللون
- إخفاء الألوان التي نفذ مخزونها (totalStock <= 0)
- تعطيل زر اللون عند نفاد المخزون

#### ج. عرض الكمية بجانب كل مقاس:
- حساب إجمالي الكمية لكل مقاس (عبر جميع الألوان)
- عرض الرقم باللون الأخضر أسفل اسم المقاس
- إخفاء المقاسات التي نفذ مخزونها
- تعطيل زر المقاس عند نفاد المخزون

#### د. تحسينات بصرية:
- **للألوان**: 
  - خط أحمر قطري على الألوان غير المتوفرة
  -_opacity_ و _cursor: not-allowed_ للألوان غير المتوفرة
  -Tooltip يوضح الكمية المتوفرة
  
- **للمقاسات**:
  - خلفية رمادية للمقاسات غير المتوفرة
  - نص باهت ومؤشر عدم السماح
  - عرض الكمية بخط صغير أخضر

## 🎯 النتيجة النهائية

### للعملاء:
1. **عرض انتقائي**: يظهر فقط الألوان والمقاسات المتوفرة
2. **معلومات واضحة**: الكمية مكتوبة بجانب كل خيار
3. **تجربة أفضل**: لا يمكن اختيار خيارات غير متوفرة

### للإدارة:
1. **إدخال أسهل**: كل توليفة (لون + مقاس + كمية) في صف واحد
2. **تحكم أفضل**: إدارة دقيقة للمخزون
3. **معاينة محسنة**: عرض واضح للتوليفات في لوحة التحكم
